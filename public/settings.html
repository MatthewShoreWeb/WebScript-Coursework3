Display Project

Hello, welcome to my application. The purpose of this application is to display the status of a project to a team of programmers. This panel is the control panel for the unattended display.

This application is fully scalable and should work on all mobile and desktop devices.

Above there are buttons which enable the user to change the display of the application in order to improve the accessibility of the application. There is also a button which enables the user to edit the content of the unattended display from this control panel. Finally, there is a button which takes you to the unattended display in a new tab.

Please click these links to confirm the HTML & CSS validate.

Website Font Scaling:

You can choose which font you would like the application to use by selecting one of the web-safe fonts from the drop down menu. This feature was added to improve accessibility/usability of the application.

You can choose the colour-scheme of the application by selecing one of the pre-set schemes from the drop down menu. All of these colour schemes are designed for people who are colour-blind and because they are an easy to see combination. This improves the accessibility/usability of the application.

You can use the slider to choose the size of the website font. This can make the website easier to read, improving the overall accessibility/usability of the application. Move the slider to the right to increase the font size and move the slider to the left to decrease the font size.

Here you can enter the information which you would like to send to the unattended display. Below is a guide to inputting valid input.

Project Name: "Any valid string" Max length 60 characters.

Project Client: "Any valid string" Max length 60 characters.

Project Members: "Any valid string" Max length 140 characters.

Project Name: Standard date format: dd/mm/yyyy cannot exceed days per month or month per year.

Project Languages: "Any valid string" Max length 60 characters.

When you have entered your information click the submit button to send the information to the unattended display. Note: all input fields must be full/valid to send the information.
